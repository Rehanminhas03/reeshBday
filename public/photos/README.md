# 📸 Drop your photos & videos here

This folder feeds the **DNA Photo Helix** centerpiece. The site reads **every
image AND video in this folder automatically** — you don't edit any code.
Videos appear as cards too, and quietly **play when they swing to the front**
of the helix (muted + looping), then pause again as they rotate away.

Supported video: `.mp4`, `.webm`, `.mov`, `.m4v`. Keep them short & muted-
friendly; portrait (9:16 / 3:4) frames best.

## How to add your pictures

1. Drop image files in this folder.
2. Name them so they sort the way you want them to appear, e.g.:
   ```
   photo-1.jpg
   photo-2.jpg
   photo-3.jpg
   ...
   ```
   (Files are sorted naturally, so `photo-2` comes before `photo-10`.)
3. Refresh the page. That's it.

## Recommended

- **Aspect ratio:** 3:4 **portrait** (e.g. 900 × 1200 px). The cards are
  portrait, so portrait photos fill them best.
- **Format:** `.jpg`, `.png`, `.webp`, or `.avif`.
- **Count:** designed for **8–12 photos**, but any number works.
- **Size:** keep each under ~500 KB if you can, for fast loading.

## Before you add photos

Until you drop real images here, the helix shows soft **placeholder cards** so
you can see the whole experience working immediately.
